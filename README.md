# Embedded Projects
# Embeddedprojects-
Collection of Embedded Systems projects including IoT, 8051 microcontroller, and VLSI design implementations.
## 1. Industrial Fault Monitoring System using IoT
Developed a real-time industrial fault monitoring system using ESP32 and ThingSpeak cloud. The system monitors machine parameters and detects faults, improving efficiency and reducing downtime.

## 2. GPS Interfacing with 8051 Microcontroller
Designed a GPS-based location tracking system using 8051 microcontroller. Implemented UART communication to receive GPS data and display real-time location on an LCD.

## 3. Power and Delay Reduction using MTCMOS Technique
Implemented a VLSI design technique using MTCMOS to reduce leakage power and delay in digital circuits, improving overall performance.
